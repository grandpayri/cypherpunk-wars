LEGACY_CREDITS

Cypherpunk Wars is a non-profit tribute developed for the Kaspa Hackathon 2026. This project is a modern reimagining of Archmage, the legendary PBBG from the late 90s. We aim to bring its deep strategic gameplay into the era of proof-of-work decentralization.

LEGAL_NOTICE: This work is a transformative homage. All code, assets, and branding are original to Cypherpunk Wars. We are not affiliated with the original Archmage developers.
