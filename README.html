3. Conflict Resolution (Attacks)

The Protocol: Attacks are recorded as public Challenges. The Enforcement Engine (SPA) verifies the attack history during BOOT and applies State-Diffs to the Defender's local memory before their next save.

vProgs Transition: In Phase 2, this will move to Multi-Writer AccessMetadata, where the winner's proof automatically updates the global state root.
